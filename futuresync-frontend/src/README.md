# 🚀 FutureSync

FutureSync is a full-stack productivity intelligence platform that helps users monitor their daily productivity, analyze performance trends, predict future productivity, and receive personalized recommendations for improving consistency and discipline.

---

## ✨ Features

- 🔐 JWT Authentication (Login & Registration)
- 👤 User Management
- 📊 Productivity Dashboard
- 📈 Analytics Dashboard
- 🤖 Future Productivity Prediction
- 💡 Smart Recommendation System
- 📅 Activity Tracking
- 📉 Performance Visualization
- 🗄️ MySQL Database Integration
- 🌐 REST APIs using Spring Boot

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Framer Motion
- Axios

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA

### Database
- MySQL

## 📂 Project Structure

FutureSync
├── futuresync (Spring Boot Backend)

---

## 🔄 Workflow

1. User Registration
2. User Login using JWT Authentication
3. Dashboard displays productivity metrics
4. Analytics visualizes study history
5. Prediction engine estimates future productivity
6. Recommendation engine suggests improvements
