# 🚀 FutureSync

FutureSync is a full-stack productivity intelligence platform that helps users monitor their daily productivity, analyze performance trends, predict future productivity, and receive personalized recommendations.

## ✨ Features

- User Registration & Login (JWT Authentication)
- Productivity Dashboard
- Activity Tracking
- Analytics Dashboard
- Prediction Engine
- Smart Recommendations

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- JPA / Hibernate

### Database
- MySQL
